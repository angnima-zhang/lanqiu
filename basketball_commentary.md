# 篮球比赛文字播报

## 使用说明

- 共 100 条基础动作播报；跟随描述或玩梗属于额外内容，不计入基础动作数量。
- A01—A50 为首批动作，均配有 B 类跟随句；C01—C50 为新增动作，仅在适合时配有 D 类可选跟随句。
- 跟随句不是必选内容，游戏程序应根据对应编号决定是否触发。
- `{{player}}`：当前执行动作的球员。
- `{{defender}}`：主要防守球员。
- `{{teammate}}`：配合进攻的队友。
- `{{team}}`：当前球员所属球队。
- `{{opponent_team}}`：对方球队。
- `{{coach}}`：当前球队主教练。
- 程序可单独展示动作句；存在对应跟随句时，也可按 `A01 → B01` 或 `C01 → D01` 连续展示。
