# 《寻道大千》视觉特效触发与类型参考

## 文档说明

公开资料可以确认《寻道大千》的玩法和实机画面，但未发现官方发布的完整特效事件表。本文根据公开实机、商店介绍和玩法资料整理，并将内容分为：

- **玩法确定**：公开资料能够确认该事件或系统存在。
- **实机归纳**：根据公开试玩画面归纳出的表现方式。
- **设计建议**：适合同类游戏的特效方案，不代表官方内部配置。

这里讨论的是触发逻辑和视觉语言。不要直接提取、描摹或复制《寻道大千》的原始序列帧、贴图、角色动画与 UI 素材。

## 五、特效与音效的同步规则

| 事件结构 | 视觉峰值 | 声音峰值 | 推荐关系 |

| 挥砍→命中 | 木屑喷出的第一帧 | 木击瞬态 | 完全同帧 |

## 六、性能与可读性规则

1. **高频事件限制粒子数**：自动砍树、普通攻击和连续资源到账不能反复制造全屏粒子。
2. **同类特效设置并发上限**：普通命中建议最多同时存在 3–5 组，超出后复用或跳过。
3. **重要信息高于装饰**：属性比较、伤害数字和状态图标不能被光柱与粒子遮挡。
4. **屏震按事件分级**：普通攻击不震或极轻，暴击轻震，Boss 击破和重大成长才能使用中度屏震。
5. **避免高频全屏白闪**：白闪仅用于渡劫、极稀有揭示等低频事件，并提供“减少闪烁”选项。
6. **自动模式降低强度**：自动挂机时粒子数量可降低 30%–60%，重复演出可加速或跳过。
7. **特效应使用对象池**：命中粒子、上浮数字、资源飞行图标等高频对象应复用。
8. **低端设备提供档位**：关闭扭曲、模糊和大面积半透明叠加，保留关键信息层。
