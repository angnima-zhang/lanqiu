- [x] 球队等级与斗志升级曲线
  - 文件：`data/balance/team_progression.json`
  - 已确定：球队市值共130级；每级市值包含4个球队等级小段，分别对应季后赛首轮、次轮、分区决赛、总决赛，共520级；每次招募固定获得10斗志；斗志升满当前市值小段后进入待夺冠；夺冠后市值+1并解锁下一组球队等级；市值提升后球队等级不重置。

- [x] 预算经济闭环
  - 文件：`data/balance/economy.json`
  - 已确定：招募固定消耗1预算并获得10斗志；预算来源为在线挂机、逐场比赛奖励、离线挂机和直接预算广告；在线基础1预算/分钟，离线基础20预算/小时且最多累计8小时；单场胜利基础预算为 `ceil(对手OVR/696)`，失败不给奖励；夺冠不再额外发预算；五个管理层共用1–520级升级成本曲线。

- [x] 赛季难度曲线
  - 文件：`data/balance/season_pacing.json`
  - 已确定：赛季难度采用竞品式固定数值表并与球队市值解耦；完成98–0后自动开启下一赛季，难度依次为新秀、饮水机、轮换、第六人、首发、核心、全明星、最佳阵容、MVP、FMVP、名人堂、传奇、GOAT；每档总决赛门槛为该品质单卡OVR上限×12；常规赛1–82场保持线性增长，第1–41场为全明星前、第42–82场为全明星后；季后赛四轮按85%/90%/95%/100%严格递增；下一品质赛季第1场必须高于上一赛季总决赛；运行时不根据玩家阵容或预期成长动态重算。

- [x] 比分生成参数
  - 文件：`data/balance/season_pacing.json`
  - 已确定：比赛先按OVR规则判定胜负，再生成比分；比分由双方12人得分属性平均值×5×0.1生成，允许随GOAT/概念神数值膨胀到几万或几亿；分差按胜方原始得分比例和OVR差距率生成，五五开区间最多拉开胜方原始得分的12%；若原始分数与已判定胜负冲突，则抬高胜方分数；四节按25%±7%权重拆分并保证四节总和等于全场总分。
