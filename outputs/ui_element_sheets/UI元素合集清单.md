# UI元素合集清单

## 制作规格
- 输出：8张PNG合集图。
- 背景：纯白。
- 内容：独立摆放、互不重叠、无说明文字。
- 边缘：每个元素带细黑色像素描边。
- 风格：与现有深青黑、青色、金色、橙红色篮球经营界面一致。
- 品质色：绿、蓝、紫、橙、红、粉、金、虹彩。

## 文件
| 文件 | 内容 |
|---|---|
| `01_通用框体按钮控件.png` | 按钮、框体、标签页、进度条、滑杆、开关、复选框、下拉框、滚动条、状态徽标 |
| `02_球员卡品质框头像环.png` | 8色小卡框、中卡框、大卡框、头像环、锁定卡、选中框、最低OVR状态框 |
| `03_主界面管理层功能图标.png` | 篮球、奖杯、球馆、招募、五类管理层、资源、图鉴、设置等 |
| `04_比赛HUD结算组件.png` | 计分板、节比分框、场地图、播报框、播放控制、胜负弹窗、战力比较 |
| `05_招募开卡比较组件.png` | 球探道具、卡背、剪影卡、开卡光效、权重行、比较卡、五项属性图标 |
| `06_赛季成就奖章组件.png` | 13品质奖章、奖杯、桂冠、赛季节点、进度条、冠军框、彩纸 |
| `07_球馆收益广告奖励组件.png` | 球馆、离线场景、预算、宝箱、广告、领取按钮、通知和奖励图标 |
| `08_设置辅助功能组件.png` | 音频、画面、辅助、输入设备、安全区、保存、重置和通用功能图标 |

## 使用提示
- 白底可按连通区域或近白色阈值批量去除。
- 球员品质小图只使用色框，不使用品质文字。
- 大球员卡可在运行时叠加品质名称、球员名、OVR和属性数字。
- 按钮与面板建议作为视觉参考重新做九宫格切片，避免直接缩放导致像素边缘变形。
