# UI 元素合集：画风匹配细拆版

输出目录：`outputs/ui_element_sheets_style_matched/final/`

这版以现有游戏界面图为画风参考，保留暗青面板、青色描边、金色主按钮、像素黑边、篮球经营题材图标和轻微漫画化质感。

## 拆分原则

- 进度条拆成：空框、填充条、分段块、端帽、刻度、指针、圆形仪表零件。
- 按钮拆成：可复用按钮背景 + 独立 icon。
- 球员卡拆成：卡壳、头像窗、品质边框、星/徽章、铭牌底板。
- HUD/奖励拆成：记分牌框、状态格、控制按钮底板、奖励图标、招募卡框等独立件。
- 不把文字、数字、球员头像、icon 和底板强行烙在一起。

## 使用建议

- 面板、按钮、进度条框建议做 9-slice。
- 进度条在游戏内组合：`空框` + `裁剪后的填充条` + 可选 `端帽/刻度/指针`。
- 按钮在游戏内组合：`按钮底板` + `独立 icon` + 游戏内字体文字。
- 球员小图组合：`头像` + `品质色框`；品质文字只在球员详情/大卡中渲染。
