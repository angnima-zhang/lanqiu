# 缺失界面补全清单

## 交付规格
- 基准分辨率：390 × 844
- 方向：竖屏
- 风格：深青黑像素篮球经营 UI、青色结构描边、金色主操作、橙红风险提示
- 参考：现有主界面与比赛准备界面
- 生成方式：OpenAI 内置图像生成，逐界面独立生成

## 本轮不制作
- 联盟：原型标记为后续社交入口，不属于当前 MVP 闭环。
- 概念神专属界面：专属颜色、卡框、卡面和招募动画仍未定稿。

## 生成提示词共同约束
- 390×844 竖屏高保真像素游戏 UI。
- 沿用现有深青黑、青色描边、金色主按钮和橙红风险色。
- 中文信息层级清楚；关键文字有描边或暗色面板。
- 关键操作处于约5%安全区内，按钮为大触控目标。
- 状态不只依赖颜色，同时使用图标、形状和文字。
- 不使用真实联盟、球队、赞助商或品牌标志。
- 不使用水印，不采用照片写实风格。
