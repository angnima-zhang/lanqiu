# UI 元素细拆版清单

输出位置：`outputs/ui_element_sheets_refined/`

本版原则：
- 白色背景，元素之间留足空隙，方便后续抠图。
- UI 形状尽量拆成可复用零件：框、底板、填充、icon 分离。
- 按钮类不把 icon 烙进背景；同一按钮底板可复用不同 icon。
- 进度条类不把进度值烙进条内；空框、填充条、端帽、分段块、刻度、指针分开。
- 球员品质不写文字，小图只用品质色框/星/徽章表达；大卡文字应由游戏内字体渲染。

## 推荐切图方式

- 可拉伸的面板、按钮、进度条框建议做 9-slice。
- 进度条实现建议：`空框` + `裁剪后的填充条` + 可选 `端帽/刻度/指针`。
- 按钮实现建议：`按钮底板` + `icon` + 游戏内文字；不要把文字或 icon 固化到按钮背景里。
- 球员小头像实现建议：`头像图` + `品质色框`；品质文字只在球员详情/大卡中由 UI 字体渲染。
